# Регистрация бота в MAX

1. Откройте в MAX чат с **@MasterBot**.
2. Выполните `/create` и следуйте подсказкам:
   - имя бота;
   - username (заканчивается на `_bot`).
3. Получите **Access Token** — это ваш `MAX_BOT_TOKEN`.
4. Включите меню команд через `/commands` и пропишите (копируйте блок целиком):

   ```
   start - Главное меню и приветствие
   menu - Открыть меню с кнопками
   search - AI-поиск: /search <запрос>
   subscribe - Подписка на мониторинг: /subscribe <тема>
   unsubscribe - Отписаться: /unsubscribe <тема>
   list - Мои подписки
   settings - Настройки (глубина поиска, статистика)
   sources - Статус источников новостей
   help - Справка
   ```

5. Webhook — см. `docs/DEPLOY.md`.

## Inline-кнопки

Бот отправляет **клавиатуру под сообщением** (поиск, подписки, настройки, обновить дайджест).
Для работы кнопок webhook должен получать события `message_callback` — это уже указано
в `setWebhook` (`update_types` в `src/lib/max.ts`).

## Если API MAX отличается

Документация MAX Bot API на момент написания минимальна. Если реальные имена
методов или формат payload отличаются — правьте `nextjs-bot/src/lib/max.ts` и
при необходимости схему кнопок в `src/lib/bot/menu.ts`.
